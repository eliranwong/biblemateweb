# **Notepad Guide**

This tool provides a local notepad for taking personal Bible study notes in Markdown format, with the ability to preview formatted output and download your notes.

## **What is the Notepad?**

The Notepad is a local note-taking tool that allows you to write Bible study notes in Markdown format. Your notes are stored locally in your browser and can be downloaded as files. It's perfect for quick notes, study outlines, or personal reflections that you want to keep on your device.

## **How to Use the Notepad**

1. **Open the Notepad**: Select "Notepad" from the tools menu.
2. **Start Typing**: The notepad opens in Edit Mode, ready for typing.
3. **Format with Markdown**: Use Markdown syntax to format your notes.
4. **Switch to Read Mode**: Click "Read Mode" to see the formatted preview.
5. **Save or Download**: Use the toolbar buttons to manage your notes.

## **Edit Mode**

In Edit Mode, you can:
- Type freely in the text area
- Use Markdown syntax for formatting
- Edit existing content
- Copy and paste text from other sources

The text area is:
- Full-height for maximum writing space
- Auto-focused when you open the tool
- Automatically saved in your browser

## **Read Mode**

In Read Mode, your Markdown is rendered as formatted HTML:
- Headers become styled headings
- Lists are properly formatted
- Links are clickable
- Tables are displayed beautifully
- Bible references become clickable links

**Tip**: Switch to Read Mode to see the formatted result before downloading or copying your notes.

## **Markdown Formatting**

The notepad supports standard Markdown syntax:

### **Headers**

```markdown
# Heading 1
## Heading 2
### Heading 3
```

### **Emphasis**

```markdown
*italic text*
**bold text**
***bold and italic***
```

### **Lists**

```markdown
- Bullet point 1
- Bullet point 2
  - Nested bullet

1. Numbered item 1
2. Numbered item 2
```

### **Links**

```markdown
[Link text](https://example.com)
```

### **Tables**

```markdown
| Column 1 | Column 2 |
| -------- | -------- |
| Data 1   | Data 2   |
```

### **Code Blocks**

```markdown
\`\`\`
Code block
\`\`\`
```

### **Bible References**

Simply type Bible references, and they'll become clickable in Read Mode:
```
John 3:16
Genesis 1:1-3
Romans 8:28-39
```

## **Features**

### **Local Storage**

- Notes are automatically saved in your browser
- No internet connection required
- Notes persist across sessions
- Stored locally on your device only

### **Download and Import**

**Download**:
- Saves notes as `biblemate_notes.md`
- Markdown format preserves all formatting
- Can be opened in any text editor or Markdown viewer

**Import**:
- Upload `.md` or `.txt` files
- Previously downloaded notes can be imported
- Overwrites current content (be careful!)

### **Clear Function**

- Confirmation dialog prevents accidental deletion
- Completely erases all content
- Cannot be undone

### **Bible Reference Parsing**

The notepad automatically recognizes and formats Bible references:
- References become clickable in Read Mode
- Click a reference to open it in the Bible area
- Supports standard reference formats (book chapter:verse)

### **Extended Markdown Support**

The notepad includes enhanced Markdown features:
- **Tables**: Full table support with alignment
- **Fenced Code Blocks**: Syntax-highlighted code
- **Table of Contents**: Auto-generated TOC
- **Cuddled Lists**: Lists without blank lines between items

## **Tips for Effective Use**

1. **Outline Structure**: Use headers to organize sermon outlines or study notes.
2. **Quick Notes**: Keep the notepad open while reading to jot down thoughts.
3. **Bible Journaling**: Record insights verse by verse.
4. **Study Preparation**: Draft teaching materials or small group questions.
5. **Reference Linking**: Type Bible references to create quick-access links.
6. **Regular Downloads**: Download important notes regularly as backup.
7. **Switch Views Often**: Toggle between Edit and Read mode to check formatting.

## **Example Workflows**

### **Sermon Note-Taking**

1. Open Notepad alongside the sermon Bible passage
2. Use headers for main points:
   ```markdown
   # Sermon on Romans 8
   ## Point 1: Life in the Spirit
   - Key verse: Romans 8:1
   - Application: ...
   ```
3. Switch to Read Mode to review formatted notes
4. Download for future reference

### **Personal Bible Study**

1. Create a study structure:
   ```markdown
   # Study of Psalm 23
   ## Observations
   - The Lord is my shepherd
   - I shall not want

   ## Questions
   - What does "shepherd" mean in this context?

   ## Applications
   - Trust God's provision
   ```
2. Click Bible references to verify verses
3. Download completed study

### **Prayer List**

1. Organize prayer requests:
   ```markdown
   # Prayer List - January 2024
   ## Family
   - [ ] Mom's health
   - [ ] Brother's job search

   ## Church
   - [ ] Youth ministry growth
   ```
2. Check off items as prayers are answered
3. Keep track of God's faithfulness

## **Data Privacy**

- Notes are stored only in your browser's local storage
- No data is sent to any server
- Notes are not backed up automatically
- You are responsible for downloading important notes
- Clearing browser data will delete your notes

## **Keyboard Shortcuts**

Most keyboard shortcuts work in the text area:
- **Ctrl/Cmd + A**: Select all
- **Ctrl/Cmd + C**: Copy
- **Ctrl/Cmd + V**: Paste
- **Ctrl/Cmd + Z**: Undo
- **Ctrl/Cmd + Y**: Redo

## **Troubleshooting**

**Notes disappeared:**
- Check if you accidentally clicked "Clear"
- Browser data may have been cleared
- Solution: Always download important notes as backup

**Import not working:**
- Ensure file is `.md` or `.txt` format
- Check that file isn't corrupted
- Try copying content manually instead

**Formatting not showing:**
- Make sure you're in Read Mode
- Check your Markdown syntax
- Preview formatting before downloading
