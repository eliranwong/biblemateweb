# Custom Token

Custom token is designed for access control to custom resources and AI features.

## To Developer

To set up a custom token, specify a string as the value of `config.custom_token` in the configuration file `~/biblemate/biblemateweb.config`.

## To Users

To use the custom token, provided by your server administrator:

1. Obtain a token from your server administrator
2. Go to `Preferences`
3. Under the section `User & Custom Data`
4. Enter a valid token in the field `Custom Token`
